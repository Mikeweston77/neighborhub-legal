📍 STEP 1: Create GitHub Repository
────────────────────────────────────
1. Go to: https://github.com/new
2. Repository name: neighborhub-legal
3. Visibility: PUBLIC (required for free GitHub Pages)
4. Do NOT check any initialization options
5. Click "Create repository"

📍 STEP 2: Push Your Code
────────────────────────────────────
Copy your repository URL from GitHub, then run:

    cd "/Users/mike/Desktop/Waterfall 3 V1.06"
    
    git remote add origin https://github.com/YOUR-USERNAME/neighborhub-legal.git
    
    git branch -M main
    
    git push -u origin main

📍 STEP 3: Enable GitHub Pages
────────────────────────────────────
1. Go to your repo → Settings → Pages (left sidebar)
2. Under "Source":
   - Branch: main
   - Folder: /docs
3. Click "Save"
4. Wait 1-2 minutes

📍 STEP 4: Get Your URLs
────────────────────────────────────
Your documents will be live at:

    https://YOUR-USERNAME.github.io/neighborhub-legal/privacy.html
    https://YOUR-USERNAME.github.io/neighborhub-legal/terms.html

📍 STEP 5: Add to App Store Connect
────────────────────────────────────
When submitting your app:

Privacy Policy URL: https://YOUR-USERNAME.github.io/neighborhub-legal/privacy.html
Support URL:        https://YOUR-USERNAME.github.io/neighborhub-legal/
